\# Парсер расписания электричек



Программа для парсинга расписания электричек с сайта Туту.ру. Извлекает информацию о рейсах электричек и позволяет фильтровать их по дням недели.



\## Описание проекта



Программа выполняет парсинг HTML-страницы с расписанием электричек и извлекает следующие данные:

\- \*\*Время отправления\*\* - в формате ЧЧ:ММ

\- \*\*Маршрут\*\* - направление следования в формате "Откуда → Куда"

\- \*\*Дни следования\*\* - "будни" или "ежедневно"



Реализована фильтрация рейсов по типу дней. Результаты сохраняются в JSON-файл и выводятся в консоль.



\## Источник данных



HTML-страница скачивается с сайта Туту.ру и сохраняется локально как `schedule.html`. Парсинг выполняется из локального файла, что обеспечивает стабильность работы независимо от изменений на live-сайте.
