# Парсер расписания электричек



Программа для парсинга расписания электричек с сайта Туту.ру. Извлекает информацию о рейсах электричек и позволяет фильтровать их по дням недели.



# Описание проекта



Программа выполняет парсинг HTML-страницы с расписанием электричек и извлекает следующие данные:

\- Время отправления - в формате ЧЧ:ММ

\- Маршрут - направление следования в формате "Откуда → Куда"

\- Дни следования - "будни" или "ежедневно"



Реализована фильтрация рейсов по типу дней. Результаты выводятся в консоль.



# Источник данных



Парсит актуальное расписание прямо с сайта Туту.ру. Не требует сохранения HTML-страницы локально. Всегда показывает самое свежее расписание.



# Использование (команды)
Все рейсы:
python parse_sputnik.py

Только будни:
python parse_sputnik.py --filter будни

Только ежедневные:
python parse_sputnik.py --filter ежедневно
